Release notes and update history for MTGA+. Each update has its own page — handy for sharing.

Deck Folders and Multi-Select v0.17.0
May 2026
Multi-select decks with Ctrl / Shift-click, bulk delete with one confirm, and user folders that pin above MTGA's built-in My Decks / Starter Decks sections — right-click to rename or delete, drag decks in to organize your collection however you want.
Latest
Custom Game Editor Update v0.15.0
May 2026
Define your own custom game modes from a web editor: pick the legality source (Scryfall query or set/rarity filter), set rarity caps, deck size, sideboard scope, and a per-mode Discord webhook. Plus an in-game searchable format dropdown, multi-select server browser filters, and full support for Omenpath / Universes-Beyond cross-printings.
